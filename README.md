# Netflix UI Clone

Single Page Application cloning Netflix's UI, with couple changes.

## Technologies:

- React
- React Router
- Styled-Components
- React-Hook-Form
- Yup
- TMDb (The Movie Database) API

## Installation

To run locally on development mode, you'll need Node.js installed. Then simply clone this repo and:

```sh
$ npm install
$ npm start
```
